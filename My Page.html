Important Concepts

Lesson 1: The Basics of the Web and HTML

How the Web Works

The Web is a complex system that connect different computers to share a multitude of content - documents, videos, audio, etc - This computers could be divide into client computers and server computers, that interchange files through a communication protocol, HTTP.

HTML

HTML stands for Hypertext Markup Language.HTML has a set of instructions that tell the browser how to render the text in the document.

Tags and Elements

HTML documents are made of HTML elements. When writing a HTML, we tell the browsers the type of each element by using HTML tags.

Why Computers are Stupid

Computers are stupid because the follow the instructions that are given to them without questions. This is a double-edge sword, on the one hand they perform precisely what was indicated, but on the other hand if the instructions are not correct they carry them out without noticing the errors.

Inline vs Block Elements

HTML elements are either inline or block elements. Block elements have an invisible box around them, this means they always start in a new line and take up the whole width of the available; while, inline elements don't start in a new line and only take as much width as needed.

Lesson 2: Creating a Structured Document with HTML

Developer Tools (in the Browser)

Today, all of the major browsers have a series of features that help the developers better understand how they are rendering the HTML and all the other documents associated with it.

The "tree-like structure" of HTML

The "tree-like structure" comes from the fact that HTML elements can have other elements inside of them. You can draw this relationship like a family tree. My mother had multiple children. So did her mother, and so on...

In a browser, this structure shows up as a series of nested boxes. There are boxes inside of boxes inside of boxes, and so on...

The relationship between indented HTML and boxes

When you read an HTML document as text, you see a wave of changing indentations going up and down the left side of the document. The more indented an element is, the more deeply nested it's corresponding "box" is.

Text Editors (for programming)

When writing code, programmers use special text editors (like Sublime Text for example). These editors make the programmer's life easier. For example, some text editors will automatically generate a closing HTML tag when you write an opening tag.
